# Customer Sales Analysis

**Tools Used:** Excel (Pivot Tables, Charts, Formulas)

**Project Description:**
- Cleaned and organized sales data for customers.
- Calculated total spending using formulas.
- Created Pivot Tables to summarize totals.
- Designed charts to visualize spending.
- Derived key insights: top spender, lowest spender, and total sales.

**Skills Gained:** Data cleaning, Pivot Tables, Charts, Data Analysis
